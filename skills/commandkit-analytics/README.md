# commandkit-analytics skill

This skill covers analytics integration for CommandKit via PostHog and Umami providers.

## Use this skill when

- adding analytics to a bot
- instrumenting command or plugin events
- reducing/no-op telemetry for specific operations

## Typical inputs

- analytics provider choice
- event naming/measurement goals
- privacy constraints

## Expected outputs

- provider wiring in config
- event instrumentation strategy
- safe payload guidance
